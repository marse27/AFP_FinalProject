\title{\textbf{AFP Project:} An Ownership Type System with Borrowing and Lifetimes}
\author{Cosmin Marian}

\begin{document}

\maketitle

This report describes the design and implementation of a Rust-inspired language
with an ownership type system, built as a Haskell type checker and interpreter.
The language prevents use-after-move errors via affine (non-copyable) types,
enforces memory safety through immutable and exclusive mutable borrows, supports
non-lexical lifetime analysis for precise borrow expiry, allows lifetime-annotated
functions that safely return references, and provides a type-safe \texttt{spawn}
construct for concurrent blocks \cite{klabnik_understanding_2021}.
The implementation uses four required Haskell techniques---a persistent scope
stack, van Laarhoven lenses, monad transformer stacks, and concurrent
primitives---and is validated by 291 passing tests
\cite{weirich_implementing_2023}.
